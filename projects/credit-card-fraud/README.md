# Credit Card Fraud Detection System

Welcome to the Credit Card Fraud Detection project!

This folder contains screenshots of my Credit Card Fraud Detection System developed using Machine Learning.

## Project Overview

The Credit Card Fraud Detection System is designed to identify fraudulent credit card transactions by analyzing transaction details using Machine Learning algorithms. The application helps distinguish between genuine and fraudulent transactions, supporting financial security and fraud prevention.

## Project Features

- Credit Card Fraud Prediction
- Secure Transaction Analysis
- Machine Learning-Based Classification
- User-Friendly Interface
- Fast and Accurate Predictions
- Fraud Detection in Real Time

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Project Workflow

1. Load the credit card transaction dataset.
2. Clean and preprocess the transaction data.
3. Train the Machine Learning model.
4. Predict whether a transaction is fraudulent or legitimate.
5. Display the prediction through an interactive Streamlit interface.
