# Profile Photo

This folder contains the profile picture used in my personal portfolio website.

## File

- `kashithra.jpeg` – My profile photo displayed on the Home section of the portfolio.

## HTML Reference

```html
<img src="images/profile/kashithra.jpeg"
     alt="Kashithra J"
     class="profile-image">
```

## CSS Class

```css
.profile-image {
    width: 220px;
    height: 220px;
    border-radius: 50%;
    object-fit: cover;
    border: 5px solid #1565c0;
}
```
