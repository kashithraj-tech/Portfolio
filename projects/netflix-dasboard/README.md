# Netflix Dashboard

Welcome to the Netflix Dashboard project!

This folder contains screenshots of my interactive Netflix Dashboard developed using Microsoft Power BI. The dashboard provides insights into Netflix's content library through data visualization and business intelligence techniques.

## Project Overview

The Netflix Dashboard is designed to analyze Netflix movies and TV shows using interactive visualizations. It helps users explore content distribution, ratings, genres, release years, countries, directors, and other key insights through an easy-to-use dashboard.

## Dashboard Features

- Total Movies and TV Shows
- Content Distribution by Type
- Genre Analysis
- Rating Analysis
- Release Year Trends
- Country-wise Content Distribution
- Top Directors Analysis
- Interactive Filters and Slicers
- KPI Cards
- Interactive Charts and Visualizations

## Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Cleaning
- Data Visualization
