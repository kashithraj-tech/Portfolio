# Spam SMS Detection System

Welcome to the Spam SMS Detection project!

This folder contains screenshots of my Spam SMS Detection System developed using Machine Learning and Natural Language Processing (NLP).

## Project Overview

The Spam SMS Detection System is designed to classify SMS messages as either **Spam** or **Ham (Not Spam)**. The application analyzes the text of incoming messages using Machine Learning techniques and predicts whether the message is legitimate or unwanted spam.

## Project Features

- Spam SMS Classification
- Real-Time Message Prediction
- Natural Language Processing (NLP)
- Machine Learning-Based Classification
- User-Friendly Streamlit Interface
- Fast and Accurate Predictions

## Project Workflow

1. Load the SMS dataset.
2. Clean and preprocess the text messages.
3. Convert the text into numerical features using TF-IDF Vectorization.
4. Train the Machine Learning classification model.
5. Predict whether the SMS is Spam or Ham.
6. Display the prediction through an interactive Streamlit interface.
