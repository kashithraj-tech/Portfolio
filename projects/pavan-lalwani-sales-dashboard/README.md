# Power BI Sales Dashboard

Welcome to the Power BI Sales Dashboard project!

This folder contains screenshots of my interactive Sales Dashboard created using Microsoft Power BI.

## Project Overview

The Sales Dashboard provides a comprehensive analysis of sales performance using interactive visualizations. It helps users understand sales trends, customer behavior, product performance, and business growth through meaningful insights.

## Dashboard Features

- Sales Overview
- Revenue Analysis
- Profit Analysis
- Customer Analysis
- Product Performance
- Regional Sales Analysis
- Interactive Filters and Slicers
- KPI Cards
- Charts and Visualizations

## Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
