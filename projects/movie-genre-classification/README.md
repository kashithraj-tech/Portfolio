# Movie Genre Classification System

Welcome to the Movie Genre Classification project!

This folder contains screenshots of my Movie Genre Classification System developed using Machine Learning and Natural Language Processing (NLP).

## Project Overview

The Movie Genre Classification System predicts the genre of a movie based on its title and description. It uses Natural Language Processing techniques to analyze text and classify movies into their appropriate genres.

## Project Features

- Movie Genre Prediction
- Text Preprocessing
- Natural Language Processing (NLP)
- Machine Learning Classification
- User-Friendly Interface
- Fast and Accurate Predictions

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Streamlit

## Project Workflow

1. Load the movie dataset.
2. Clean and preprocess the text data.
3. Convert text into numerical features using TF-IDF.
4. Train the Machine Learning model.
5. Predict the movie genre.
6. Display the prediction through a Streamlit interface.
